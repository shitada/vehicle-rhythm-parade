# 🚆 のりものリズムパレード

4〜5歳向けのリズムタップゲーム。乗り物ごとの固有リズムに合わせてタップしよう！

## 🎮 あそぶ

👉 **https://shitada.github.io/vehicle-rhythm-parade/**

## 📖 あそびかた

1. **「あそぶ」をタップ** してゲームスタート
2. まず **おてほん** が流れるので、リズムを覚えよう（タップ不要）
3. **「さあ はじめよう！」** が出たら本番スタート
4. **金色に光ったオーブ** → タップ！ ✅
5. **赤色に光ったオーブ (💤)** → タップしない！ ❌
6. リズムに乗って連続ヒットを狙おう

## 🚗 ラウンド構成

### 通常ラウンド（R1〜R10）

10種類の乗り物にはそれぞれ固有のリズムパターンがある。

| 乗り物 | リズムの特徴 |
|--------|------------|
| 🚆 でんしゃ | ガタンゴトン — 規則正しい2拍リズム |
| 🚌 バス | ゆらゆら — スイングリズム |
| 🚒 しょうぼうしゃ | ウーウー — 3連打サイレン |
| 🚕 タクシー | スイスイ — シンコペーション |
| 🚢 ふね | ざぶーん — おやすみオーブ初登場 🔴 |
| 🚁 ヘリコプター | ブーンブーン — プロペラ回転 |
| 🚓 パトカー | ピーポー — タップと おやすみの交互 |
| 🚜 ブルドーザー | ぐいぐい — ピストン運動 |
| ✈️ ひこうき | スーッ — 高速交互パターン |
| 🚀 ロケット | ビューン — 最高速の交互+罠 |

### ボーナスステージ（R11〜R15）

通常ラウンドをクリアすると、複数の乗り物のリズムがランダムに切り替わるミックスステージが登場！

| ステージ | 内容 |
|---------|------|
| 🏙️ まちの のりもの | でんしゃ+バス+タクシー |
| 🔧 はたらく くるま | しょうぼうしゃ+パトカー+ブルドーザー |
| 🌤️ そらの のりもの | ヘリ+ひこうき+ロケット |
| 🌟 オールスター | 全10種からランダム |
| 🎪 ファイナル パレード | 全10種ランダム×最高速 |

## ⭐ リワード

- **3連続ヒット** → スター獲得 ⭐
- **5連続ヒット** → フィーバーモード 🎉
- おやすみオーブをタップすると連続ヒットがリセット
